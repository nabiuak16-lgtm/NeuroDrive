# NeuroDrive

AI-powered driver safety ecosystem — premium mobile application demo.

## Features

- **NeuroScore** — real-time driver safety score (0–100)
- **Drive Mode** — live monitoring with AI risk predictions
- **Analytics** — trends, charts, and AI insights
- **Trip History** — detailed trip reports
- **Device Center** — Smart Wheel, Smart Watch, phone sensors
- **AI Coach** — personalized safety recommendations
- **Emergency Mode** — critical condition alerts
- **Breathing Exercises** — stress relief guidance
- **Achievements** — gamification system
- **Demo Mode** — realistic simulated data (no hardware required)

## Run

```bash
cd NeuroDrive
npm install
npx expo start
```

Scan the QR code with Expo Go (iOS/Android) or press `w` for web.

## Tech Stack

- React Native + Expo SDK 57
- TypeScript
- React Navigation
- Reanimated 4
- Expo Blur, Linear Gradient, Haptics

## Language

100% Russian UI.
